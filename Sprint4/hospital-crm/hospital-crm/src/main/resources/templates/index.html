Documentação da API (JSON Endpoints)

Pacientes
  • GET /api/pacientes - Listar todos
  • GET /api/pacientes/{id} - Buscar por ID
  • POST /api/pacientes - Cadastrar paciente
  • GET /api/pacientes/{id}/imc - Calcular IMC
Leads
  • GET /api/leads - Listar todos
  • GET /api/leads/{id} - Buscar por ID
  • POST /api/leads - Cadastrar lead
Agendamentos
  • GET /api/agendamentos - Listar todos
  • GET /api/agendamentos/{id} - Buscar por ID
  • POST /api/agendamentos - Agendar consulta
  • PATCH /api/agendamentos/{id}/status?status=valor - Atualizar status
